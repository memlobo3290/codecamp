1. ค้นหาข้อมูลว่าแท็ก <!doctype> มีหน้าที่อะไร
ใช้บอกให้เว็บเบราเซอร์ทราบว่า เราใช้คำสั่ง HTML รุ่นใด และบอกชนิดของเอกสาร (Document Type Definition : DTD) ที่ใช้ ซึ่งจะช่วยให้เว็บเบราเซอร์แปลเอกสารได้อย่างถูกต้อง

2. ถ้าอยากให้มี tooltip ใน tag นั้นต้องใช้ attribute ใด
-ใช้ attribute title

3. แท็ก <b>,<i> ต่างจาก <strong>,<em> หรือไม่อย่างไร
<b> กับ <strong> ทำให้เป็นตัวหนาเหมือนกัน แต่ strong ใช้เน้นเน้นข้อความที่สำคัญมาก สำคัญใน SEO
<i> กับ <em> ทำให้เป็นตัวเอียงเหมือนกัน แต่ <em> เป็นการระบุข้อความที่เน้น

4. หา tag ที่สามารถแสดงข้อความกลับหลังได้
<bdo dir="rtl"> กลับหลัง </bdo>
