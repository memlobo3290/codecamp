ให้หาว่า input type มีอะไรบ้างแต่ละ type แตกต่างกันอย่างไร
Type Description
button กำหนดปุ่มคลิกได้ (ส่วนใหญ่ใช้กับ JavaScript เพื่อเปิดใช้งานสคริปต์)
checkbox กำหนดช่องทำเครื่องหมาย
color กำหนดตัวเลือกสี
date กำหนดตัวควบคุมวันที่ (ปี, เดือน, วัน (ไม่มีเวลา))
datetime-local กำหนดการควบคุมวันที่และเวลา (ปี, เดือน, วัน, เวลา (ไม่มีเขตเวลา)
email กำหนดข้อมูลสำหรับที่อยู่อีเมล (ต้องใส่เป็นรูปแบบ email)
file กำหนดฟิลด์เลือกไฟล์และปุ่ม "Browse" (สำหรับการอัปโหลดไฟล์)
hidden กำหนดฟิลด์อินพุตที่ซ่อนอยู่
image กำหนดรูปภาพเป็นปุ่มส่ง
month กำหนดการควบคุมเดือนและปี (ไม่มีเขตเวลา)
number กำหนดเขตข้อมูลสำหรับการป้อนตัวเลข
password กำหนดฟิลด์รหัสผ่าน
radio กำหนดปุ่มตัวเลือก
range กำหนดตัวควบคุมช่วง (เช่นตัวควบคุมแถบเลื่อน)
reset กำหนดปุ่มรีเซ็ต
search กำหนดฟิลด์ข้อความสำหรับป้อนสตริงการค้นหา
submit กำหนดปุ่มส่ง
tel กำหนดเขตข้อมูลสำหรับการป้อนหมายเลขโทรศัพท์
text ค่าเริ่มต้น. กำหนดฟิลด์ข้อความบรรทัดเดียว
time กำหนดตัวควบคุมสำหรับการป้อนเวลา (ไม่มีเขตเวลา)
url กำหนดเขตข้อมูลสำหรับการป้อน URL
week กำหนดการควบคุมสัปดาห์และปี (ไม่มีเขตเวลา)
