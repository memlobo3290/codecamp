หาความเหมือนและความแตกต่าง ระหว่าง 2 tag ที่กำหนดให้
ชื่อ Tag ความเหมือน ข้อแตกต่าง
<hr> vs <br> เว้นบรรทัดเหมือนกัน tag <hr> จะมีเส้นขีดเพิ่มขึ้นมา แต่ของ <br> ไม่มี
<ul> vs <ol> เป็น list เหมือนกัน tag <ul> จะไม่เป็นตัวเลข เป็นเครื่องหมายแทน แต่ของ <ol> จะเป็นตัวเลข/ตัวอักษร/เลขโรมัน เรียงลำดับ
<select> vs <checkbox> vs <radio button> เป็น form ให้เลือก tag <select> เป็น drop-down list
tag <checkbox> เป็น form ให้เลือกติ๊กถูก แบบสี่เหลี่ยม
tag <radio button> เป็น form ให้เลือกแบบวงกลม
<div> vs <span> เหมือน block ของข้อมูลต่าง tag <span> span นิยมใช้สำหรับข้อความธรรมดา เอาไว้ใส่ stylesheet

ตัวอย่างการใช้งาน
<span> A </span>
<span> B </span>
<div> C </div>
<div> D </div>
ผลที่แสดง จะเป็นดังนี้
A B
C
D
<script> vs <style> เอาไว้กำหนดองค์ประกอบ tag <style> ใช้เพื่อกำหนดข้อมูลสไตล์สำหรับเอกสาร HTML
tag <script> ใช้เพื่อกำหนดสคริปต์ฝั่งไคลเอ็นต์ (JavaScript)
